# URL (https://www.youtube.com/)

The first component of a url is protocol (https:// :- hyper text transfer protocol secure)

The second component of a url is domain (www.youtube.com) it is a user friendly name of ip address of my server

The third component of a url is path(/ :- home)
example of some paths are (/, /about, /project/tic-tac-toe)

The fourth component of a url is querry parameters (https://www.youtube.com/search?user=manoj+kumar+patel)
the values after ? are querry parrameters (we can send any key value pair here)
