# HTTP Methods

there are mainyl 5 types of methods
1. Get
2. Post
3. Put
4. Patch
5. Delete

## Get
when you want to get some data from the server 
by default browser always do a **get request**

## Post
when we want to send and mutate some data in the server can be used on html forms

usually we use just get and post 

put :- when we need to put some data on our server like uploading a file or img to server
patch :- when we need to change a existing entry
delete :- when need to delete the data

can use req.method method(function) to get the method type of req
