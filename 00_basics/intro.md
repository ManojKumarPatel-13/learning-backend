# Node JS

### What is Node JS?
Node JS is a runtime environment for js 
c++ and v8 engine of chrome are embeded and made node js

### How to setup Node JS?
To setup node js on mac you can use brew or directly download from site

lts -> stable one (long time support) 

node js follow odd even approach even is lts and odd one is latest

commands to check if node is installed or not 
``` Command
node -v
npm -v
```
